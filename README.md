# Functionalities
- registration
- login / logout
- reset password
- save / update / remove post
- OAuth 2.0 / OIDC
- sending confirmation / reset emails
- search for user's posts
- search for posts by content
- pagination
- form validation

# Used Tools
- Java
- HTML / CSS / JS
- Thymeleaf
- Spring Security
- Spring Data JPA
- Hibermate
- Spring Cache / Ehcache
- Java Mail Sender
- Liquibase
- Testcontainers
- JUnit5 / Mockito
- MySQL / PostgreSQL
